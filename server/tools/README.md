# MongoDB Database Tools - Instalación Local

## Pasos para configurar:

1. **Descomprime** MongoDB Database Tools en esta carpeta
2. **Renombra** la carpeta principal a `mongodb-tools`
3. **Estructura final:**
   ```
   server/
   ├── tools/
   │   └── mongodb-tools/
   │       ├── bin/
   │       │   ├── mongodump.exe
   │       │   ├── mongorestore.exe
   │       │   ├── mongoexport.exe
   │       │   └── mongoimport.exe
   │       └── LICENSE
   └── ...
   ```

## Uso:
Los scripts usarán automáticamente las herramientas desde esta carpeta.
